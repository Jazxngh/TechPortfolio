Autodesk

For my 1st year software project module, I collaborated with Jack from Autodesk to design and develop a multi layered 3D Asset Management System using C#. The purpose of the software is to allow organisations to store, categorise, and manage 3D assets like characters, buildings, and objects, with options for both public and private visibility. To make sure we achieve Industry standard for usability, we implemented a role based system demostrating different features for regular users, managers, and system administrators. Through this project, we successfully was able to achieve industry standard requirements into a software solution capable of handling large scale organisation process.

Velociti

For my 2nd year software project module otherwise known as professional software project, I collaborated with a group of 4 developers with Leon to design and develop a website for Velociti using ReactJS, Javascript, Socket.IO and leaflet.JS as well as testing with both manual testing and other tests like Jest. This is a railway travel based website where you can track and manage your train journeys from train times, locations and routes using velociti's API securely. In this case, to make sure we achieve industry standards for usability and performance by we implementing a design that is accessible but also intuitive to meet the broader user needs as well as functions that improves performance such as superclustering, loading and so forth
